🔗 เชื่อมต่อ Google Sheets (Apps Script)

วาง URL ของ Google Apps Script Web App ที่ Deploy แล้ว (Execute as: Me, Who has access: Anyone)

📋 Google Apps Script — คัดลอกโค้ดนี้

1. เปิด Google Sheets → Extensions → Apps Script → วางโค้ดด้านล่าง → Deploy → Web App

หลัง Deploy: ตั้ง Execute as: Me และ Who has access: Anyone แล้ว copy URL มาวางด้านบน

🔐 จัดการรหัส PIN

PIN ใช้ป้องกันการเข้าใช้งาน app โดยไม่ได้รับอนุญาต
ระบบจะล็อกอัตโนมัติเมื่อไม่ใช้งาน 10 นาที

PIN ผู้ใช้ — สำหรับพนักงานบันทึกสต็อกทั่วไป
PIN Admin — สำหรับเจ้าของร้าน ใช้เข้าหน้าตั้งค่าและเปลี่ยนรหัส
PIN เริ่มต้น: ผู้ใช้ = 1234 | Admin = 9999

📱 วิธีใช้งานบนมือถือ

วิธีที่ 1 — บันทึกไฟล์ใช้ออฟไลน์
1. ดาวน์โหลดไฟล์ stock_app.html ไปไว้ในมือถือ
2. เปิดด้วย Chrome / Safari แล้วกด Add to Home Screen
3. จะได้ icon บนหน้าจอ กดเปิดได้เลย (ไม่ต้อง internet)

วิธีที่ 2 — เปิดผ่าน URL (แนะนำถ้ามีหลายมือถือ)
1. อัปโหลดไฟล์ขึ้น GitHub Pages หรือ Google Drive
2. แชร์ URL ให้พนักงาน เปิดด้วย Chrome แล้ว Add to Home Screen
3. ทุกคนใช้ URL เดียวกัน อัปเดตครั้งเดียวใช้ได้ทุกเครื่อง
